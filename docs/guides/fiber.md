# Fiber moment-curvature

The `sap2000py.fiber` package discretizes a section into fibers and computes
moment-curvature response under axial load, with Mander confined/unconfined
concrete and bilinear steel constitutive models. Requires the `fiber` extra.

!!! note
    Implemented in milestone M3. This page will document the constitutive
    models, section discretization, and the `MomentCurvature` solver.
